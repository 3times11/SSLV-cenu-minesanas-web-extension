# SS.lv cenu minēšanas spēles Web plugin

## Apraksts
Web extension, kurš neļauj lietotājiem redzēt sludinājumu cenas, kā arī pārējo informāciju par sludinājumiem, to pārskatīšanas lapā.
**Pilnu informāciju par sludinājumu var redzēt tikai tad, kad atver konkrēto sludinājuma lapu.**
**Cenu sludinājumam var atklāt tikai tad, kad uzspiež uz teksta "Uzspied šeit, lai redzētu cenu." sludinājuma lapā.**

## Ieteikumi lietošanai
Šis Web extension vislabāk strādās, kad to izmantos uz **Chromium balstītiem interneta pārlūkiem (Chrome, Brave utml.).**
Šo Web extension ir iespējams arī izmantot uz Firefox balstītiem pārlūkiem, bet nezināmu iemeslu dēļ, Firefox pārlūkā cenu dati no 
sludinājumiem netiek izdzēsti pietiekami laicīgi, tādējādi bojājot cenu minēšanas spēles pieredzi visiem.

## Extension uzstādīšana Chrome interneta pārlūkā
1. Dodieties uz "chrome://extensions/" lapu.
2. Spiediet "Load unpacked" pogu un dodieties uz ielādēto "SSLV-cenu-minesanas-web-extension" mapi.
3. Kad esat nonākuši šajā mapē, spiediet "Select Folder" (ja esat uz Windows). Kad šo būsiet izdarījis, extension būs automātiski ieslēgts
jūsu pārlūkā un SS.lv mājaslapā cenas un informācija par sludinājumiem būs apslēpta.

Ja vēlaties izslēgt šo extension, tad dodaties atkal uz "chrome://extensions/" lapu un izslēdziet plugin "SSlv listing price guessing".
